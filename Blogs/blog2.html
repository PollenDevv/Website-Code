How to get out of tutorial hell

When starting programming it's easy to think that programming is just about typing words on a screen, but when in reality that is only a small part, the main part of programming is problem solving and design choices.

So when you start programming it can be very overwhelming, so what do you do? You look up a tutorial or follow a guidebook, this is fine but the problem comes when you keep using tutorials, again and again. At what point are you following a tutorial instead of programming yourself, it also takes away trying to complete a solo project by yourself.
